ファイルの作成日時と更新日時をexifデータやファイル名に従って修正するプログラム

## 使い方

    update_fstat.py: ファイルのタイムスタンプをexifやファイル名に従って更新する
    
    Usage:
        update_fstat.py <fnames>...
        update_fstat.py --version
    
    Options:
        <fnames>     : filenames
        -h --help    : show this message
        --version    : show version
